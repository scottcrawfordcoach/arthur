# Exploring Contemporary Issues
It‚Äôs important for the ICF Code of Ethics to remain current and
relevant as the profession continues to grow and mature. For
this reason, we initiate a review of the Code every three years to
address contemporary issues and ensure consistent alignment and
understanding for global coach practitioners.
In collaboration with the Code Review Steering Committee, we
began the latest review in mid-2023. Their work was focused on
data collection, through surveys and interviews with stakeholders
across the ICF ecosystem and getting a Code Review Team
established to draft an updated Code of Ethics.
Work on the Code review is ongoing and anticipated to conclude
